MUI.Themes.Symlinker is based on Symlink Creator with the goal to a better UI. You can get the program here. The older version and the changelog are here.

P.-S.: Firefox needs an addon to use ClickOnce: Get the addon for Firefox.
Also, unfortunately, Chrome doesn't support support ClickOnce and no addon is available for it, but you can use any other browser to install it (Internet Explorer, Mozilla Firefox, Safari, Opera, etc...)
P.-P.-S.: You may want to install this certificate so that the User Account Control and ClickOnce recognizes me as a trusted editor. How to: Open the file, click on "Install the certificate". Press on next, put all the certificates in the following store: "Trusted Root Certification Authorites" or the second, for nationalized ones. Click next, then finish. Approve the security advertissement and press OK. UAC and ClickOnce now recognizes the signature! The signature will break if the program is modified, so you will know if the program has been modified.
